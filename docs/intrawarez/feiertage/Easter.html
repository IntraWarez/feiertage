static int gauss(int $year)

Computes the "Gauss-Day-Number" of Easter Sunday for a given year.

This is the modern style implementation of the Gaussian Easter Algorithm taken from here and here.

The Gaussian Easter Algorithm computes the number of the day of easter sunday as n-th day of March, i.e. 32th March == 1st April.

Parameters

Return Value

int The "Gauss-Day-Number" of Easter Sunday for the given year.

static DateTime date(int $year)

Computes the date of Easter Sunday for a given year.

Instead of relying on PHP's native easter_date function this implementation is based on the Gaussian Easter Algorithm, so it is not limited to unix years, i.e.: 1970 - 2037.
